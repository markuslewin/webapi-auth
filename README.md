# Auth using Minimal API and ASP.NET Core Identity

```sh
aspire run
```

Browse to the [Swagger UI](https://localhost:7185/swagger/index.html).

Reimplements custom ASP.NET Core Identity endpoints as seen in [`IdentityApiEndpointRouteBuilderExtensions.cs`](https://github.com/dotnet/aspnetcore/blob/main/src/Identity/Core/src/IdentityApiEndpointRouteBuilderExtensions.cs).
